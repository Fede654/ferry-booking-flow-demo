# Ferry booking flow demo

An interactive concept demo of an availability-first ferry booking flow.

## Important

This is not an official Herjólfur booking service. Availability, prices, the reservation timer, account prefill, and payment are client-side demonstration data only. No booking can be made through this site.

## What it demonstrates

- Lightweight route and date search
- Passenger and multi-vehicle capacity decisions before sailing selection
- Release-and-reselect protection when a change invalidates held sailings
- Preserved traveller details during backward navigation
- Responsive mobile checkout frame with an editable search context

The repository intentionally contains only the runnable demo assets; internal research notes, test suites, and reconstructed baseline material are excluded.
